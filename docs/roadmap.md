# RealityCheck AI Roadmap

## Phase 1: MVP Completed

- FastAPI backend
- React/Vite/Tailwind frontend
- GitHub project creation
- Repository cloning/fetching
- Deterministic scanners
- Optional LLM reasoning
- Gap storage
- Severity scoring
- Reality Score
- Markdown report export
- Premium dashboard, filters, detail view, architecture graph
- Local demo repository with intentional mismatches

## Phase 2: Product Improvements

- Deeper AST parsing for Python, TypeScript, and framework-specific route graphs
- Vector DB repository memory for historical context and semantic docs comparison
- LangGraph multi-agent orchestration for scanner, reviewer, recommender, and reporter roles
- PR patch generation for docs/config fixes
- GitHub App integration with PR comments and repository installation flow
- Historical scan comparison and trend charts
- CI/CD integration to fail builds on high-severity drift
- Slack/Jira export for team workflows
- Team workspace support with projects, members, and scan history
- Hosted production deployment with PostgreSQL, background jobs, and queue-based scanning
