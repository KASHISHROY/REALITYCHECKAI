# Resume Bullets

## Concise Version

- Built RealityCheck AI, a full-stack repository intelligence platform that detects documentation drift, API mismatches, environment gaps, dependency inconsistencies, and deployment reality gaps.

## Technical Version

- Implemented a FastAPI, SQLAlchemy, React, TypeScript, and Tailwind system with deterministic scanners for FastAPI/Express routes, frontend API calls, env vars, Docker/Compose config, dependency signals, auth mechanisms, severity scoring, persisted gaps, and markdown report export.

## Startup/Product Version

- Designed and shipped an MVP SaaS-style engineering tool that turns repository drift into an actionable dashboard with Reality Score, severity filters, gap details, architecture graph, demo mode, and shareable reports.

## GenAI-Focused Version

- Architected a hybrid deterministic-scanning plus GenAI-reasoning pipeline that uses exact static analysis for repository facts and optional Groq/OpenAI agents for documentation claim extraction, engineering explanations, and fix recommendations.
